# stats/reports/

Ez az almappa a nem runtime-jellegu reportok gyujtohelye.

A cel az, hogy a jelenlegi `stats/` gyokerszinten keveredo compliance, batch, cleanup es
domain-specific anyagok kesobb ide rendezhetok legyenek, kis, kovetheto cleanup-korokben.

Aktualis allapot:
- az elso tenylegesen athelyezett report-csoport mar ide kerult
- a canonical runtime batch summaryk most mar a `batches/` almappaban vannak
- a tobbi report-csoport egyelore meg a `stats/` gyokerszinten maradt
